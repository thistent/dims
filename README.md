# DIMS : A decentralized platform for idea formation and DAO experimentation with a built-in algorithmic mechanism for monetary stability without the need for an external oracle.

## Problem
Current funding mechanisms suffer from the following problems:
 - Incentives don't encourage collaboration 
 - Autonomy is sacrificed when voting is necessary
 - They aren't designed for grassroots DAO experiments

## Solution
An open-source graph-based knowledge management tool featuring:
 - Asynchronous composable workflows
 - Flexible funding strategies
 - An internal pegging mechanism for a median-wage-based algorithmic stable coin

See my progress [here](https://thistent.github.io/dims/)!
